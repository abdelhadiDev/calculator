# Requirements

Utilisation de Symfony 5.2 <br>
PHP 7.4

## Installation

```bash
git clone git@github.com:abdelhadiDev/calculator.git

Use Symfony server

composer install
symfony serve -d

or 

use docker
docker-compose up
go to http://calulator.local:8888/calculator
```
